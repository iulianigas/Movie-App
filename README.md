# Movie-App

Lucrarea de licenta, o aplicatie web de gestionarea a filmelor pe care am dezvoltat-o utilizand tehnologiile: Ruby & Ruby on Rails, HTML, CSS, JavaScript, Bootstrap si jQuery

<p align="center"> Fiecare film are mai multe review-uri acordate de catre utilizatori. Un utilizator ofera un review filmului/filmelor pe care acesta le-a vizualizat. Review-urile acordate sunt de la 1 la 5 in lilieci. Nu se pot edita sau sterge review-urile altor utilizatori. Descrierea filmului/filmelor poate fi editata doar filmului/filmelor pe care utilizatorul le-a adaugat in aplicatie, iar de asemenea filmul/filmele adaugate de acesta pot fi sterse </p>
